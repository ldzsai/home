# 公司官网首页

展示公司基本信息和形象的官方网站首页。

## 技术栈

- Vue 3
- TypeScript
- Tailwind CSS
- Vite
- Font Awesome

## 开发

### 依赖安装

```bash
pnpm install
```

### 本地开发

```bash
pnpm run dev
```

### 构建

```bash
pnpm run build
```

### 预览生产构建

```bash
pnpm run preview
```

## 部署

项目通过 GitHub Actions 自动部署到 GitHub Pages。
